## Introduction
This repo is what powers the helix website.

## Developing
Install the Helix CLI:

    sudo npm install -g @adobe/helix-cli
    
Run it in this repo:

    hlx up
    
## Testing
### Install dependencies:

    npm install
### Run tests:

    npm test
### Watch tests:

    npm test:watch
    
This will run through all your tests, but then only run against any changed tests.
