‚ö†Ô∏è Core Web Vitals Interpretation Guidelines

When analyzing LCP, CLS, and INP metrics:

  • If metric titles contain "and rising" or "and falling", these indicate TRENDS, not absolute performance
  • "Rising" means the metric is getting worse over time, "falling" means it's improving
  • DO NOT conclude the site is bad based solely on "rising" trends - consider the actual metric values
  • Focus on the current metric values and their impact, not just the trend direction
  • A metric can be "rising" but still within acceptable performance thresholds

  • classes: "performance score-ni" = Needs Improvement - Metric is borderline. Mention if relevant but not critical.

  • classes: "business-metric acquisition bounces" - Bounce rate metric. If value is above 60%, this is a CRITICAL ISSUE indicating serious retention problems. Values below 40% are healthy.
  • classes: "business-metric acquisition organic" - Organic traffic/acquisition metric. Low values may indicate SEO or discoverability issues.
  • Note: The actual value is in <span class="formatted-value"> nested inside <number-format>, and <meter> attributes (low, high, optimum) provide threshold guidance
